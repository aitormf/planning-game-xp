# Revisión de PR con IA

La IA revisa todas las Pull Requests.

## Rol de la IA
- Analiza el diff
- Detecta problemas
- Emite OK o NOK

## Rol del humano
- Toma la decisión final
- Evalúa contexto y producto

## Política
- IA OK + Humano OK = merge permitido
- IA NOK = la PR no puede mergearse
